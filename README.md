## Development

### requirements

- Python 3.10+
- [PDM](https://pdm.fming.dev/latest/) (Python Dependency Management)

Checkout the repository:

```bash
git clone git@github.com:yuchanns/nastool.git
cd nastool
git submodule update --init --recursive
```

Install dependencies and run dev:
```bash
pdm install

pdm run dev
```
